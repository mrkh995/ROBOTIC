# آموزشگاه رباتیک آریما — ROBOTIC

وب‌سایت رسمی **آموزشگاه رباتیک فنی و حرفه‌ای آریما** در کرج؛ آموزش پروژه‌محور رباتیک، الکترونیک، برنامه‌نویسی و هوش مصنوعی برای کودکان، نوجوانان و بزرگسالان.

## فناوری‌ها

- **Next.js 15** (App Router) + **React 19**
- **TypeScript**
- **Tailwind CSS v4** (با `@theme` و توکن‌های اختصاصی برند)
- **lucide-react** برای آیکون‌ها
- **react-hook-form** + **zod** برای فرم مشاوره

## اجرای پروژه

```bash
npm install
npm run dev
```

سپس آدرس `http://localhost:3000` را باز کنید.

## ساختار پوشه‌ها

- `app/` — صفحات و مسیرها (App Router)
- `components/ui/` — اجزای پایه‌ی رابط کاربری
- `components/blocks/` — بخش‌های صفحه (هیرو، ناوبری، فرم و ...)
- `lib/data/` — داده‌های سایت، دوره‌ها و دسته‌بندی‌ها
- `lib/` — توابع کمکی، آیکون‌ها و اعتبارسنجی

- زبان سایت فارسی و جهت صفحه راست‌به‌چپ (RTL) است.
- فونت پیش‌فرض **Vazirmatn** است.
- اطلاعات تماس (تلفن، ایمیل، اینستاگرام) هنوز نهایی نشده و با برچسب «در حال تکمیل» نمایش داده می‌شود.
